Welcome to BloomWatch! 🎉

A comprehensive platform for monitoring and visualizing plant
blooming events worldwide using NASA Earth observation data.

🚀 QUICK START

1. Run Setup:
   ./setup.sh

2. Activate Environment:
   source venv/bin/activate

3. Choose Your Path:
   
   📡 A) Backend API:
      cd backend && python app.py
   
   📓 B) Jupyter Notebook:
      jupyter notebook notebooks/bloom_analysis.ipynb
   
   🌐 C) Web Interface:
      xdg-open frontend/index.html

    • Sentinel-2 (10m)

    • SAVI (Soil-Adjusted VI)
    • GNDVI (Green NDVI)

🔍 Smart Bloom Detection

    • Change detection algorithms
    • Peak identification
    • Duration tracking
    • Confidence scoring

📊 Rich Visualizations
    • Interactive global maps
    • Time series plots
    • Multi-year comparisons
    • Statistical dashboards

🤖 Predictive Analytics
    • Historical trend analysis

Backend:
  • Python 3.8+
  • Flask/FastAPI
  • NumPy, Pandas, SciPy
  • Rasterio, GeoPandas
  • earthaccess (NASA data)

Frontend:
  • HTML5, CSS3, JavaScript
  • Bootstrap 5
  • Leaflet.js
  • Chart.js

Data:
  • NASA Earthdata
  • Microsoft Planetary Computer
  • ESA Copernicus (Sentinel)

💡 TIPS FOR SUCCESS

1. ⚡ Start with Demo Mode
   No NASA credentials? No problem! The system will
   automatically generate synthetic data for testing.

2. 🗺️ Use Suggested Locations
   Start with our pre-configured bloom sites for
   guaranteed results.

3. 📓 Explore the Notebook
   The Jupyter notebook includes complete examples
   and explanations.

4. 🔄 Compare Years
   Most interesting insights come from multi-year
   comparisons. Try 2020 vs 2023!

5. 🌍 Think Global, Act Local
   The system scales from individual fields to
   entire continents.

🐛 TROUBLESHOOTING

Problem: Import errors
Solution: Ensure virtual environment is activated
         source venv/bin/activate

Problem: NASA data access fails
Solution: Check credentials in .env file or use demo mode

Problem: GDAL installation fails
Solution: Install system dependencies first
         Ubuntu: sudo apt-get install gdal-bin libgdal-dev
         macOS:  brew install gdal

See QUICKSTART.md for more help!
